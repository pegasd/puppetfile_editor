This gem provides CLI and Rake interfaces for maintaining Puppetfile.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'puppetfile_editor'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install puppetfile_editor
```
